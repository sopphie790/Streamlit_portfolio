# 🌸 About Me Portfolio Web App (Streamlit)

This is a personal portfolio web application built using Streamlit, showcasing a modern, aesthetic, and interactive design with custom CSS styling, sidebar navigation, and profile presentation.

It is created for a student developer portfolio (Liza Jaime).

---

## ✨ Features

- 🎀 Pink aesthetic modern UI design  
- ⭐ Animated floating gold star background in sidebar  
- 🧭 Sidebar navigation with custom styling  
- 🖼️ Oval 3D profile image frame design  
- 📍 Location display in sidebar footer  
- 💻 Clean and professional portfolio layout  
- 🎨 Fully customized CSS styling (buttons, fonts, layout)  
- ⚡ Fast and lightweight Streamlit web app
